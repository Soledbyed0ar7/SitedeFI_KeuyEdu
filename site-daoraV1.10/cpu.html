A CPU

Central Processing Unit - Unidade de Processamento Central

A CPU é o "cérebro" do computador. Realiza as tarefas, processos e cálculos que devem ser realizados. Embora existam outros processsadores em um computador, a CPU, como o nome sugere, é o processador principal. Não confundir com o computador completo - a CPU é apenas uma parte do computador, e é errado chamar o computador de CPU.

A velocidade da CPU, medida em GHz (MHz em processadores mais lentos), representa quantos ciclos de processamento a CPU pode executar por segundo, mas não é o único fator que determina seu desempenho: a quantidade de núcleos, a geração, o software e outros fatores também entram em jogo - Duas CPUs de mesma velocidade mas de gerações diferentes podem ter diferença na velocidade de executar tarefas, assim como uma Quad-Core é mais rápida que uma Dual-Core.

Existem alguns tipos de CPU, que podem ter entradas de soquete diferentes, assim como CPUs de diferentes gerações podem ter entradas diferentes. Ao escolher uma CPU e/ou placa-mãe, é indispensável ter certeza de que o processador poderá encaixar no soquete da placa.

Geralmente, a CPU fica localizada sob um cooler ou, no caso de dispositivos compactos como smartphones, um dissipador de calor. Isso se faz necessário pois a CPU produz um pouco de calor durante seu funcionamento normal, e se ele não for dissipado, a CPU pode ficar quente o suficiente para se danificar bem rápido.

A CPU também ajuda a controlar o hardware.
