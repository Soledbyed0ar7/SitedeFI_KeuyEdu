A Ram

Random Access Memory-Ram

A RAM (Random Access Memory - Memória de Acesso Aleatório, por vezes chamada de Rapid Access Memory - Memória de Acesso Rápido) ‚ uma memória usada para armazenar temporariamente dados e instruções que estão sendo usados pelo processador (CPU).

A RAM é uma memória volátil, ou seja, precisa de energia para armazenar seus conte£dos, que são perdidos caso a energia seja perdida.

Ela é mais rápida que as memórias não-voláteis (formas de memória que não dependem da energia para reter os dados), o que a faz eficaz para armazenar dados que a CPU precisa acessar rapidamente.

Gerações de RAM mais recentes, como a DDR5, são mais rápidas e têm maior capacidade que gerações mais antigas, como a DDR4 e a DDR3. DDR significa Double Data Rate - Taxa de Dados Dupla.

A memória RAM costuma ser montada em placas RAM, que são inseridas no slot DIMM das placas-mãe. A maioria dos desktops têm múltiplos slots DIMM para permitir o aumento da capacidade de RAM, o que aumenta o desempenho do computador por diminuir o tempo de acesso de arquivos.
